FutuRaM Chat answers questions about the material, component and element composition of products in the FutuRaM knowledge graph. It turns your question into a SPARQL query over the flat fq: query vocabulary and runs it against the selected backend (each backend is an independent SPARQL endpoint — pick one above):

    • {% endfor %}

Ask, for example, how much of an element / material / component is in a given class — the assistant builds the fq:contains pattern, declares the constituent kind via the ontology (rdfs:subClassOf futuram:Element), and reads back the amount.
